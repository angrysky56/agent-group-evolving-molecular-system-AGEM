A Mapping Instability
The graph suggests a subtle problem neither Chalmers nor Block fully addresses:

Chalmers' "easy problems" ≈ Block's A-consciousness — approximately the same functional class
Chalmers' "hard problem" ≠ Block's P-consciousness — because Chalmers asks why A-states are accompanied by P, while Block merely describes the twoThis is a non-trivial difference. Block's distinction is taxonomic (there are two kinds of consciousness); Chalmers' is explanatory (one kind needs explaining via the other). If Block's P/A distinction is accepted, the hard problem changes shape — it becomes: "What explains the correlation between P and A?" rather than "Why is there P at all?" The former is substantially more tractable.
