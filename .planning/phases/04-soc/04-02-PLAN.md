**Create `src/soc/correlation.ts`** — pure math utility functions:

1. `pearsonCorrelation(x: number[], y: number[]): number`
   Standard Pearson r = (sum((x_i - xbar)(y_i - ybar))) / (sqrt(sum((x_i - xbar)^2)) \* sqrt(sum((y_i - ybar)^2))). Return 0 if denominator is 0 or if n < 2.

2. `linearSlope(values: number[]): number`
   Compute slope of least-squares linear fit for values indexed 0..n-1. slope = pearsonCorrelation(indices, values) * (stddev(values) / stddev(indices)). Or simpler direct formula: slope = (n*sum(i*y_i) - sum(i)*sum(y_i)) / (n\*sum(i^2) - (sum(i))^2).

**Create `src/soc/SOCTracker.ts`:**

Class `SOCTracker extends EventEmitter`:

Constructor:

- Accepts optional `Partial<SOCConfig>` with defaults: `correlationWindowSize: 10`, `surprisingEdgeSimilarityThreshold: 0.3`, `trendWindowSize: 5`.
- Private fields: `#history: SOCMetrics[]`, `#config: SOCConfig`, `#previousCorrelation: number | null`.

`computeAndEmit(inputs: SOCInputs): SOCMetrics`:

1. Compute Von Neumann entropy: call `vonNeumannEntropy(inputs.nodeCount, inputs.edges)`.
2. Compute embedding entropy: call `embeddingEntropy(Array.from(inputs.embeddings.values()))`.
3. Compute CDP: `vonNeumannEntropy - embeddingEntropy`.
4. Compute surprising edge ratio:
   - Filter `inputs.newEdges` where `createdAtIteration === inputs.iteration`.
   - For each edge, check BOTH criteria:
     a. Structural: `communityAssignments.get(source) !== communityAssignments.get(target)` (cross-community)
     b. Semantic: `cosineSimilarity(embeddings.get(source), embeddings.get(target)) < config.surprisingEdgeSimilarityThreshold`
   - Ratio = surprising count / total new edges count. 0 if no new edges.
5. Compute phase transition:
   - Push deltaS_structural and deltaS_semantic to rolling delta arrays:
     - deltaS_structural = current vonNeumannEntropy - previous vonNeumannEntropy (0 if first iteration)
     - deltaS_semantic = current embeddingEntropy - previous embeddingEntropy (0 if first iteration)
   - If delta arrays have >= `config.correlationWindowSize` entries:
     - Compute Pearson correlation over last `correlationWindowSize` entries.
     - Check sign change: `Math.sign(currentCorrelation) !== Math.sign(#previousCorrelation)` and `#previousCorrelation !== null`.
     - If sign change AND both have magnitude > 0.1 (noise filter): fire `phase:transition` event with `centeredAtIteration = inputs.iteration - Math.floor(correlationWindowSize / 2)`.
   - Update `#previousCorrelation`.
6. Build `SOCMetrics` object with all computed values.
7. Push to `#history`.
8. Emit `soc:metrics` event with full payload.
9. If phase transition detected, also emit `phase:transition` event.
10. Return metrics.

`getMetricsHistory(): ReadonlyArray<SOCMetrics>`:
Return `Object.freeze([...this.#history])` — defensive copy, same pattern as LCM ImmutableStore.

`getLatestMetrics(): SOCMetrics | undefined`:
Return `this.#history[this.#history.length - 1]`.

`getMetricsTrend(window?: number): MetricsTrend`:
Compute mean and linear regression slope of `vonNeumannEntropy` over the last `window` iterations (default: `config.trendWindowSize`). Return `{ mean, slope, window }`.
