**GREEN phase — Implement entropy functions:**

Create `src/soc/entropy.ts`:

1. **`vonNeumannEntropy(nodeCount: number, edges: ReadonlyArray<{source: number, target: number, weight: number}>): number`**

   Implementation:
   a. If nodeCount <= 1 or edges.length === 0, return 0.
   b. Build adjacency matrix A (nodeCount x nodeCount) from edges. For undirected graph, set both A[i][j] and A[j][i] to the edge weight.
   c. Build degree matrix D: `D[i] = sum of A[i][j] for all j`.
   d. Build D^(-1/2): `D_inv_sqrt[i] = 1 / sqrt(D[i])` for D[i] > 0, else 0.
   e. Build normalized Laplacian: `L_norm[i][j] = (i===j ? 1 : 0) - D_inv_sqrt[i] * A[i][j] * D_inv_sqrt[j]` for all i,j.
   f. Compute density matrix: `rho = L_norm / trace(L_norm)`. For a connected graph, `trace(L_norm) = n`.
   g. Eigendecompose rho using `math.eigs(rho)` (mathjs). Extract real eigenvalues.
   h. Compute entropy: `S = -sum(p_i * ln(p_i))` where p_i are the eigenvalues of rho, skipping p_i <= 0 (or < 1e-12 to handle numerical noise).
   i. Return S.

   Use `mathjs` for matrix construction and `math.eigs()` for eigendecomposition — same pattern as `src/sheaf/SheafLaplacian.ts`.

   **Critical design decision:** Build L_norm from the adjacency data passed in, NOT from SheafEigenspectrum. This is a different graph (TNA co-occurrence graph vs. sheaf base graph). Per RESEARCH.md Key Architecture Clarification.

2. **`embeddingEntropy(embeddings: Float64Array[]): number`**

   Implementation:
   a. If embeddings.length <= 1, return 0.
   b. Get n = embeddings.length, d = embeddings[0].length.
   c. Build covariance matrix Sigma = (1/n) * E^T * E (d x d).
      Use ml-matrix `MlMatrix` for construction: iterate over embeddings, accumulate outer products.
   d. Eigendecompose Sigma using `new EigenvalueDecomposition(Sigma)`. Extract `realEigenvalues`.
   e. Clamp negative eigenvalues to zero (numerical artifact guard, per RESEARCH.md Pitfall 5).
   f. Normalize eigenvalues: `p_i = max(0, lambda_i) / sum(all max(0, lambda_i))`. If sum is 0, return 0.
   g. Compute entropy: `S = -sum(p_i * ln(p_i))` skipping p_i <= 1e-12.
   h. Return S.

   Use `ml-matrix` for matrix construction and eigendecomposition — same pattern as `src/sheaf/CohomologyAnalyzer.ts`.

3. **`cosineSimilarity(a: Float64Array, b: Float64Array): number`** — copy the pure function from `src/lcm/LCMGrep.ts` pattern. Used by Wave 2 for surprising edge ratio. Include it here as a utility since it's needed by the SOC module and must not import from LCM.
