**RegimeValidator class:**

Purpose: Validates that a phase transition candidate (sign change in Pearson correlation) is genuine, not noise. Three conditions must ALL hold simultaneously:

1. **Persistence**: The sign change must persist for N consecutive iterations (configurable, default 3).
2. **Coherence**: Within the persistence window, the fraction of same-sign iterations must be >= threshold (default 0.6).
3. **H^1 gating**: The Sheaf H^1 dimension must be >= threshold (default 2) at the time of validation.

Implementation:

```typescript
import type {
  RegimeValidatorConfig,
} from './interfaces.js';

const DEFAULT_VALIDATOR_CONFIG: RegimeValidatorConfig = {
  persistenceWindow: 3,
  coherenceThreshold: 0.6,
  h1DimensionThreshold: 2,
};

export class RegimeValidator {
  readonly #config: RegimeValidatorConfig;

  // Rolling window of correlation sign values (+1 or -1)
  #signHistory: number[] = [];

  // Track whether a transition candidate is being validated
  #candidateStartIteration: number | null = null;
  #candidateSign: number = 0;

  // Track the last confirmed transition iteration (prevent duplicate confirmations)
  #lastConfirmedIteration: number | null = null;

  constructor(config: Partial<RegimeValidatorConfig> = {}) {
    this.#config = { ...DEFAULT_VALIDATOR_CONFIG, ...config };
  }
```

**Key methods:**

1. `trackCorrelation(correlationCoefficient: number, iteration: number): void`
   - Record the sign of the current correlation coefficient into #signHistory.
   - Trim history to 2x persistence window size (avoid unbounded growth).
   - If a sign change is detected from the previous entry, mark candidateStartIteration = iteration and candidateSign = new sign.

2. `validateTransition(h1Dimension: number, iteration: number): { confirmed: boolean; coherence: number }`
   - If no candidateStartIteration is set, return { confirmed: false, coherence: 0 }.
   - Compute the number of iterations since candidateStartIteration.
   - If fewer than persistenceWindow iterations have elapsed, return not confirmed.
   - Compute coherence: count how many of the last persistenceWindow sign values match candidateSign, divide by persistenceWindow.
   - Check all three conditions:
     a. persistence >= persistenceWindow
     b. coherence >= coherenceThreshold
     c. h1Dimension >= h1DimensionThreshold
   - If all three hold AND this iteration != lastConfirmedIteration:
     - Set lastConfirmedIteration = iteration.
     - Reset candidateStartIteration = null.
     - Return { confirmed: true, coherence }.
   - Otherwise return { confirmed: false, coherence }.

3. `getSignHistory(): ReadonlyArray<number>` — returns defensive copy of sign history.

4. `getCurrentCandidate(): { startIteration: number | null; sign: number }` — inspection helper.

**Edge cases handled:**
- Noise-induced flips: Coherence check filters brief fluctuations (if signs flip back and forth, coherence < 0.6, no confirmation).
- H^1 spikes: Even with sustained sign change, if H^1 dimension is < 2, transition is not confirmed.
- Duplicate suppression: lastConfirmedIteration prevents the same transition from being confirmed multiple times.
- Early iterations: If fewer than persistenceWindow iterations of sign history, no confirmation possible.

**Key methods:**

1. `analyzeRegime(metrics: SOCMetrics, isTransitioning: boolean): RegimeMetrics`
   - Push metrics to #metricsWindow; trim to analysisWindowSize.
   - Compute cdpVariance: variance of CDP values in the window.
   - Compute correlationConsistency: standard deviation of correlationCoefficient in the window.
   - Compute persistence: metrics.iteration - #regimeStartIteration.
   - Call #classifyRegime(cdpVariance, correlationConsistency, persistence, isTransitioning).
   - If regime changed from #currentRegime, update #currentRegime and #regimeStartIteration.

```typescript
function variance(values: number[]): number {
  if (values.length < 2) return 0;
  const mean = values.reduce((a, b) => a + b, 0) / values.length;
  return values.reduce((acc, v) => acc + (v - mean) ** 2, 0) / (values.length - 1);
}

function stdDev(values: number[]): number {
  return Math.sqrt(variance(values));
}
```
